jQuery Table SubRows

Let's face it, tables are a part of life, be it for tabular data or *gasp* design. Okay, forget I mentioned that second part. Regardless of what you use tables for, you may find yourself wanting to display subsets of data, which with tables can get to be clunky and often times an eyesore.
Enter SubRows.
SubRows will give you a lightweight javascript solution for displaying/hiding subsets of data. It provides no styles of its own -- it's css is purely for functional purposes, leaving you, the designer, with the freedom to design and display your tables however you choose. SubRows is also customizable to help you get the end result that you want.
Enjoy!

In its simplest form, SubRows needs one line of javascript, executed on the "parent" rows of your table.

Show Code
Javascript:

<script type="text/javascript">
     $("#demo1 .item").SubRows();
</script>

You have full control over SubRows. Take advantage of the options, like in this example where I customize the speed of each subrow expansion.

Show Code
Javascript:

<script type="text/javascript">
    var speed = 200;
    $("#demo2 .item").each(function(){
        $(this).SubRows({speed: speed});
        speed *= 3;
    });
</script>

Options

subrowClass - The class name that is applied to all subrow <tr> elements.

iconClass - The class name that is applied to the <span> in the parent row which contains the toggle icon

activeClass - The class applied to the iconClass when it is toggled to its active state

defaultClass - The class applied to the iconClass when it is toggled to its default state

emptyClass - The class applied to the iconClass which acts as filler space. This class is used to provide spacing consistency between rows which have subrows and those that do not.

speed - The speed at which the subrows expand and contract at. It can be either "slow", "normal", "fast" or an integer value representing the speed in milliseconds.

subRowWrapperClass - The class that is applied to the <div> which wraps the content of the subrow row.
